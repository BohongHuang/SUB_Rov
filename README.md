# JMU ROV

这是一个校园项目

使用的IDE是qt 5.14.2

是一个qml程序

依赖的库有：

gstreamer 1.0 (请将其安装于C:/gstreamer  下)



>  如果出现vc++ runtime 错误，请尝试通过安装vc运行库以修复该问题。



项目的参考对象有：

[QGroundControl][1]
